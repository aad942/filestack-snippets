# React File Upload with Progress Tracking

This project demonstrates a simple React application with file upload progress tracking using XMLHttpRequest.

## Features
- Buttons for uploading and cancelling uploads.
- Better UI handling for progress tracking
- A mock server to simulate file upload.

---

## Step 1: Set Up Your React Project

1. Create a new React app:
  
   `npx create-react-app react-progress-tracking`
   
   `cd react-progress-tracking`

---

## Step 2: Mock Backend Setup

1. Create a `server.js` file in the root directory for `server`.
2. Install dependencies:

  `npm install express multer cors`

3. Start the mock API:

   `node server.js`

---

## Step 3: Create the React Component

Inside the `src` folder, create a file named `FileUploader.js`. (Refer to `src/FileUploader.js` for the content of `FileUploader.js` file)

---

## Step 4: Update `App.js`

Replace the content of `src/App.js` with the content in the `src/App.js` file content.

---

## Step 5: Start Your App

1. Run the following command:

    `npm start`

2. Start the backend:
   
    `node server.js`

---
